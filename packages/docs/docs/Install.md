## Install

With `npm`:

```
npm install @open-ts-gen/mockgen
```

With `yarn`:

```
yarn add @open-ts-gen/mockgen
```

### TypeScript support

Open TS Gen is written in TypeScript, nothing special beyond installing the package via `yarn` or `npm` is required to
get it to work with TypeScript.
